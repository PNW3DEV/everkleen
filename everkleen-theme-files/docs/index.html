Introduction

Brainwave is theme, which is created to make impression on you and your customers. We offer you an up-to-date and really high quality work with outstanding design and well-thought usability approach. This template can help you to achieve a significant increase of the conversion rates and number of positive feedback, as well as will simply be a nice surprise for your customers.

Please do not forget to rate "Brainwave" on ThemeForest. This can be done in Downloads area. Just select "Brainwave" in the Download list and rate it. We really appreciate your feedback and will be grateful for your opinion.

Requirements For Brainwave

To use Brainwave, you must be running WordPress 3.5 or higher, PHP5.3 or higher, and mysql 5 or higher.

Below is a list of items you should ensure your host can comply with.

  • Check to ensure that your web host has the minimum requirements to run WordPress.
  • Always make sure they are running the latest version of WordPress.
  • You can download the latest release of WordPress from official WordPress website.
  • Always create secure passwords for FTP and Database.

Recommended PHP Configuration Limits

You can change php configuration on your own, or contact your web host and ask them to increase those limits to a minimum as follows:

  • max_execution_time 180
  • memory_limit 128M
  • post_max_size 32M
  • upload_max_filesize 32M

How To Install Theme

Watch "how to install theme" video

WordPress Installation

How To Install Brainwave via WordPress

  1. Navigate to Appearance > Themes.
  2. Click Install Themes and click the Upload button.
  3. Navigate to find the “Brainwave.zip” file on your computer and click “Install Now”
  4. Activate the newly installed theme by going to Appearance > Themes and clicking the Activate button.
  5. You will then see a notification message to install and activate required/recommended plugins: BW Portfolio, Contact Form 7, Visual Composer, WooCommerce.
  6. In the notification message, click the ‘Go Install Plugins’ button to go to the Install Plugins tab where you can install and activate the required & recommended plugins.
  7. Click on checkbox above each plugin checkbox, choose Install in Actions select and click on Apply button. After that they will be installed and activated automatically.

FTP Installation

How To Install Brainwave Via FTP

  1. Log into your hosting space via an FTP software.
  2. Unzip the Brainwave.zip file and ONLY use the extracted Brainwave theme folder.
  3. Upload the extracted Brainwave theme folder into wp-content > themes folder.
  4. Activate the newly installed theme by going to Appearance > Themes and clicking the Activate button.
  5. You will then see a notification message to install and activate required/recommended plugins: BW Portfolio, Contact Form 7, Visual Composer, WooCommerce.
  6. In the notification message, click the ‘Go Install Plugins’ button to go to the Install Plugins tab where you can install and activate the required & recommended plugins.
  7. Click on checkbox above each plugin checkbox, choose Install in Actions select and click on Apply button. After that they will be installed and activated automatically.

Plugin Installation

Brainwave includes 5 plugins that will may be installed and activated: BW Portfolio, WooCommerce, Contact Form 7, WP Tiles and Visual Composer. Contact Form 7, WP Tiles, WooCommerce and Visual Composer are recommended plugins. BW Portfolio is required plugin for BrainWave theme, it contain simple to use portfolio post type and many shortcodes. When you install Brainwave, you will get a notification message in your WordPress admin telling you of the recommended and required plugins.

Included Plugins With The Brainwave Theme

  • BW Portfolio – It is BrainWave theme addons plugin. Created by Us :)

  • Contact Form 7 – This plugin is free. With it you can create and customize forms for any purpose. Created by Takayuki Miyoshi
  • WP Tiles – This plugin is free. With it youcan create and customize tiles galleries. Created by Mike Martel.
  • Visual Composer – This is drug and drop page builder plugin with frontend and backend editors created by WP Bakery.

"Featured Carousel" For Posts

It's pretty simple: You click on "+ Add Image(s)" and choose images. If you want choose few images hold Ctrl button and click on images You want to add. If You want to remove image you click on cross in top right. After all images were added You must update post. That's all, enjoy carousel on post

"Display Page" For Page

Here You can choose how to display page in onepage, if You using onepage view.

  • Display Title in Onepage - defines display title for this page or not
  • Page Section - it make page section paddings smaller, normal or turn off them. For example you can use small for pages with less content, such as one little form. Also You can turn off paddings and add them for every row.
  • Show Page On Main - on/off displaying page in onepage.

"Section Background" For Page

Here You can set page background. Also You can choose background for every row on page.

  • Overlay Color - This is color of background if images not added, or overlay otherwise.
  • Overlay Opacity - Here You can choose how much transparent background color will be. For value 1 image will not be visible, for 0 color will not be visible.
  • Columns - Here You can choose how much columns will be on background. 1, 2, 3, 4 or 6 columns can be assigned.
  • Add Images - You can add image for background. You can sort them as You want. On phones visitors will see first image from list.
  • Add Empty - You can add empty background to offset next image for one column.

"Creative Header" For Page

Here You can set add creative slider that will be look like part of header (better works with large header).

  • Enable Creative Slider - Here You can turn on/off your slider.
  • Add Images - Each image will be new slide. You can sort them as You want.

"Masonry Size" For Portfolio

Here You can set size of portfolio cell in masonry grid for shortcode [portfolio] if You choose type "masonry". Available sizes: 1x1, 2x1 or 2x2.

"Categories" For Portfolio

You can add or remove portfolio categories, set or unset categories for each portfolio. To add category click on "+Add category", write name in field that will be displayed and then click on Add button below. To remove categories check them and click "-Remove category".

Warning: When removing categories, first uncheck all categories that You DON'T WANT to remove, and then check all what You want remove. You can not undo this action. If you don't sure - don't delete categories, You can simply don't use them.

Import/Export

Here You can export or import your theme settings.

There are two ways to import your settings:

  • Import from File - click and paste copied data.
  • Import From URL - click and paste copied url.

With first variant You can use data copied from other Brainwave theme settings, or from file earlier downloaded from Brainwave theme settings.

There are three ways to export theme settings: Copy Data to quickly paste in Import from File field; Download Data File for store file with current settings on hard drive as backup; Copy Export URL for paste it on Import from URL field.

Shortcodes

There are two ways use our Shortcodes:
  • Standart ( with [] ) - You can write [name-of-shortcode] and Wordpress will show exactly what You want
  • Visual Composer - just choose shortcode and customize it the way You want
Result will be the same, but with Visual Composer it's so much easier. Whatever below we will describe both of shortcode using.

[buttons]

[buttons] shortcode can help You simply add a button for different purpose on your site.

Wordpress Style

[buttons button_text="text on button" button_link="You may use button as link" button_size="btn-lg" button_type="btn-primary" icon_lib="fontawesome" icon_fontawesome="fa fa-asterisk" after_page="true"]
  • button_text - it's text that You see on button
  • button_size - four variants of size: btn, btn-lg, btn-xs, btn-sm
  • button_type - seven variants of buttons: btn-default, btn-primary, btn-success, btn-info, btn-danger, btn-warning, btn-link
  • button_link - you can put url here, and button will be used for redirect visitors to that url on click
  • icon_lib - icon library from four icon packs: fontawesome, etline, simpleline and elegant.
  • icon_fontawesome - icon from font-awesome library
  • icon_etline - icon from et-line library
  • icon_simpleline - icon from simple-line library
  • icon_elegant - icon from elegant-icons library
  • after_page - this attribute uses on many shortcodes to show current shortcode after all page content without container

[icon-tabs]

[icon-tabs] shortcode allow You add creative accordion with icons as tabs.

Wordpress Style

[icon_tabs_container][icon-tabs tab_id="Design" is_active="true" icon_lib="fontawesome" icon_fontawesome="fa fa-anchor"]Lorem ipsum, dolor sit amet.[/icon-tabs][/icon_tabs_container]

  • tab_id - Text that will be displayed in tab header.
  • is_active - Set this tab as active. Please don't set more then one tab as active.
  • icon_lib - icon library from four icon packs: fontawesome, etline, simpleline and elegant.
  • icon_fontawesome - icon from font-awesome library
  • icon_etline - icon from et-line library
  • icon_simpleline - icon from simple-line library
  • icon_elegant - icon from elegant-icons library
  • after_page - this attribute uses on many shortcodes to show current shortcode after all page content without container
